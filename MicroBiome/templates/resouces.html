Microbiome related databases

Resouces Description
Human OncoBiome Database Database of comprehensive curated information on the bacterial species present in the human cancer microbiota
MicrobiomeDB MicrobiomeDB is a data mining platform enabling the exploration of microbiome studies. This platform also provides visualization and analysis tools to investigate microbiome data.
MDB: Microbiome Database The microbiome Database gathers microbiome metadata. about the environment (oceanic, soil…) and host-associated microbiome. It also makes available the gene catalogs for host-associated microbiomes.
Human Microbiome Project Data Portal The Human Microbiome Project Data Portal enables the exploration of studies and data associated with the human microbiome. Furthermore, this web interface provides access to the tools used to analyze microbiome.
Human Oral Microbiome Database (eHOMD) comprehensive and curated information o­n the bacterial species present in the human aerodigestive tract
Disbiome database Database providing an overview of differential microbial composition in diseases
MDB: Microbiome Database This database provides sequencing resource and accurate metadata of metagenomics samples, as well as gene catalogs.
Human Pan-Microbe Communities (HPMC) database Database of human gastrointestinal microbiota derived from faecal samples.
